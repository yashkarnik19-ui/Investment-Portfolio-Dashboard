Trade Reconciliation & Risk Dashboard

Project Overview
An automated Middle-Office tool designed to identify trade discrepancies (breaks) between internal ledgers and market sources of truth. 

Key Features
* Automated Reconciliation: Processes 3,000+ tickers to find Price, Qty, and Currency breaks.
* Risk Prioritization: Calculates **Value at Risk (VaR) to highlight high-impact errors.
* Interactive Dashboard: Built with Streamlit for real-time exception management.

Tech Stack
* Python (Pandas, NumPy)
* Streamlit (UI/UX)
* Matplotlib (Risk Visualization)
